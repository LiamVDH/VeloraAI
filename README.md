# Velora AI

Velora AI is a futuristic local AI desktop assistant built with Python.

## Features

- 🤖 AI chat assistant
- 🎤 Voice input
- 🔊 Text-to-speech responses
- 🖥️ Desktop app control
- 🌐 Browser and YouTube control
- ⌨️ Keyboard automation
- 🎨 Futuristic animated interface
- 💻 Productivity assistant

## Powered By

- Python
- PySide6
- Ollama
- Qwen 2.5 AI
- Speech Recognition
